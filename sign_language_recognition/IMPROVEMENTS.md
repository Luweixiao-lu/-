# 手势识别准确率改进说明

## 已实施的改进

### 1. 特征提取改进
- **之前**: 仅使用63维相对坐标特征
- **现在**: 使用77维综合特征，包括：
  - 相对坐标特征（42维）
  - 手指长度特征（5维）
  - 手指角度特征（4维）
  - 手指伸直度特征（5维）
  - 手部中心距离特征（21维）

这些特征能更好地区分不同手势。

### 2. 模型优化
- **之前**: RandomForest (100棵树)
- **现在**: RandomForest (300棵树, 最大深度20)
- 使用分层抽样确保训练集和测试集的类别分布一致

### 3. 数据收集改进
- **之前**: 每个手势50个样本
- **现在**: 每个手势100个样本（建议）

## 重要提示

⚠️ **由于特征提取方法已改变，旧训练数据不兼容！**

您需要：
1. **删除旧数据**（或重命名备份）:
   ```bash
   rm -rf training_data/
   rm gesture_model.pkl
   ```

2. **重新收集数据**:
   ```bash
   python data_collector.py
   ```
   现在每个手势会收集100个样本（之前是50个）

3. **重新训练模型**:
   ```bash
   python train_model.py
   ```

## 预期改进效果

- 特征维度从63增加到77，包含更多区分性信息
- 模型容量增加（300棵树 vs 100棵树）
- 数据量增加（100样本/类 vs 50样本/类）
- 预期准确率提升：从可能较低的准确率提升到70-85%+

## 进一步改进建议

如果准确率仍然不够高，可以尝试：

1. **增加训练数据**: 每个手势收集200-300个样本
2. **数据增强**: 添加旋转、缩放等变换
3. **尝试其他模型**: SVM、XGBoost、神经网络等
4. **调整特征**: 根据实际效果调整特征提取方法
5. **后处理**: 使用时序平滑、置信度阈值等

# 🔧 添加模型文件到Git

## 问题

应用显示"模型未找到"，因为 `gesture_model.pkl` 文件还没有被提交到GitHub。

## ✅ 解决方案

### 方法1：直接提交（推荐，文件17MB < 100MB限制）

```bash
# 1. 添加模型文件
git add gesture_model.pkl

# 2. 提交
git commit -m "Add trained model file"

# 3. 推送到GitHub
git push
```

### 方法2：如果文件太大，使用Git LFS

如果将来文件超过100MB，可以使用Git LFS：

```bash
# 1. 安装Git LFS（如果还没安装）
# macOS: brew install git-lfs
# 或下载: https://git-lfs.github.com/

# 2. 初始化Git LFS
git lfs install

# 3. 跟踪.pkl文件
git lfs track "*.pkl"

# 4. 添加.gitattributes文件
git add .gitattributes

# 5. 添加模型文件
git add gesture_model.pkl

# 6. 提交并推送
git commit -m "Add model file with Git LFS"
git push
```

## 🎯 现在只需3步：

### 第1步：推送到GitHub

```bash
# 在项目目录下执行
git init
git add .
git commit -m "Deploy Sign Language Recognition App"

# 在GitHub创建新仓库（https://github.com/new）
# 然后执行：
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

### 第2步：部署到Streamlit Cloud

1. 访问：**https://share.streamlit.io/**
2. 用GitHub账号登录
3. 点击 **"New app"**
4. 选择您的仓库
5. 主文件路径：`app.py`
6. 点击 **"Deploy"**

### 第3步：分享链接

部署完成后（约2-5分钟），您会得到一个URL：
```
https://your-app-name.streamlit.app
```

**把这个链接分享给任何人，他们就可以直接使用了！** 🎉

1. **模型文件大小**：如果 `gesture_model.pkl` 超过100MB，可能需要使用Git LFS
2. **摄像头权限**：用户首次使用时需要授权浏览器访问摄像头
3. **浏览器推荐**：Chrome或Firefox效果最佳
